# Help Desk app - Node.js Mongodb backend

## Docker

This project supports docker and docker-compose.

Use `docker-compose up` to run application on **localhost:8080** and mongodb on port **27017**

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app.<br>
**NOTE** You have to build your app before running this command.<br>
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

### `npm run develop`

Runs the app in the development mode.<br>
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.<br>
It correctly compile latest JavaScript in to ES5 for production mode.
